# テストファイル

このファイルは Windows ファイルダブルクリック機能のテストに使用されます。

## 内容確認

- ファイル名が正しく表示されるかテスト
- ファイル内容が正しく読み込まれるかテスト
- URL パラメータが適切に処理されるかテスト

**期待される動作:**
1. ファイルダブルクリック → C# リレーアプリが起動
2. C# アプリが `http://localhost:8080/file?path=...&name=test-file.md` を生成
3. サーバーが適切にファイル内容を埋め込み
4. エディターでファイル名「test-file.md」が表示される
